# Infrastructure Learning Notes

Repository ini berisi catatan belajar tentang **Linux System Administration** dan **Networking** - dari dasar hingga implementasi.

## Tentang

Materi disusun secara sistematis dalam format tutorial dengan contoh praktis, studi kasus, dan latihan untuk memperdalam pemahaman.

**Target:** Menjadi System Administrator atau DevOps Engineer yang kompeten.

## Struktur Materi

| Bagian | Topik | Status |
|--------|-------|--------|
| **I. Fundamental OS** | Operating System, Linux Structure, Process, Permission, systemd, Filesystem | In Progress |
| **II. Fundamental Networking** | Networking Basics, IP/Subnetting, TCP/UDP, DNS/Routing, Firewall | Planned |
| **III. Integration** | Lab Project, SSH, Monitoring, Automation | Planned |
